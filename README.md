# 🌍 Lugares que Queremos Conhecer

Este projeto é um site simples desenvolvido em **HTML e CSS** como trabalho escolar.  
O objetivo é apresentar **lugares do mundo que os alunos gostariam de conhecer**, explicando os motivos e algumas curiosidades sobre cada destino.

## 📌 Sobre o projeto

**Página central contendo animações em Javascript para ter acesso as demais páginas.**

Cada aluno criou sua própria página utilizando a mesma estrutura básica de HTML.  
Em cada página são apresentadas:

- Informações sobre o aluno
- O lugar que ele gostaria de visitar
- Um pequeno texto explicando o motivo da escolha
- Algumas curiosidades sobre o local

Os destinos escolhidos incluem países e cidades conhecidos por sua **história, cultura e paisagens**, como:

- Grécia
- Roma (Itália)
- Austrália
- Argentina

## 🛠 Tecnologias utilizadas

- **HTML** – estrutura das páginas  
- **CSS** – estilização e aparência visual
- **JS** – animação de componente na tela principal

Cada aluno utilizou um arquivo CSS próprio para personalizar o visual da sua página e a página principal foi personalizada de acordo com as cores das páginas!

## 🎯 Objetivo do trabalho

O trabalho tem como finalidade praticar:

- Estrutura básica de páginas HTML
- Organização de conteúdo com **sections**
- Uso de **imagens**
- Estilização com **CSS**
- Organização de um pequeno projeto web
